Botones en Bootstrap

Los botones en Bootstrap son elementos fundamentales para interactuar con los usuarios en una aplicación web. Bootstrap proporciona una serie de clases predefinidas que permiten crear botones con diferentes estilos y colores. Estos botones pueden tener diferentes propósitos, como acciones normales, acciones destacadas, éxito, información, advertencias o peligros. Algunas de las clases de botones más comunes en Bootstrap incluyen btn-default, btn-primary, btn-success, btn-info, btn-warning, btn-danger y btn-link. Además, Bootstrap permite crear botones de diferentes tamaños utilizando las clases btn-lg, btn-sm y btn-xs, que proporcionan botones grandes, medianos y pequeños respectivamente. Para crear tu propio botón, simplemente elige una clase de botón de Bootstrap válida y agrégala al elemento <button> junto con el contenido del botón. Por ejemplo, <button class="btn btn-primary">Mi botón</button>.

Botones informativos

Los botones informativos en Bootstrap vienen en varios estilos predefinidos que se pueden usar para resaltar ciertos elementos de la interfaz de usuario. Puedes elegir entre diferentes colores para indicar diferentes tipos de mensajes:




<button class="btn btn-default">Normal</button>
<button class="btn btn-primary">Destacado</button>
<button class="btn btn-success">Éxito</button>
<button class="btn btn-info">Información</button>
<button class="btn btn-warning">Advertencia</button>
<button class="btn btn-danger">Peligro</button>
<button class="btn btn-link">Enlace</button>

Crea tu propio botón informativo ingresando una clase de botón de Bootstrap válida (por ejemplo, btn-warning, btn-danger, btn-success, btn-primary, btn-info, btn-link, etc.).

Botones de diferentes tamaños

Los botones en Bootstrap también pueden tener diferentes tamaños para adaptarse a las necesidades de diseño. Puedes elegir entre tres tamaños predefinidos:




<button class="btn btn-primary btn-lg">Grande</button>
<button class="btn btn-primary btn-sm">Mediano</button>
<button class="btn btn-primary btn-xs">Pequeño</button>
