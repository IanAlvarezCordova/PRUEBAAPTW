Grid System en Bootstrap

El Grid System de Bootstrap es un sistema de cuadrícula que permite crear diseños responsivos mediante el uso de filas y columnas. Las filas se crean utilizando la clase .row y las columnas se crean utilizando clases como .col-md-*, donde * es el número de columnas que ocupará el elemento. Además, podemos usar la clase .col-md-offset-* para desplazar columnas a la derecha.
