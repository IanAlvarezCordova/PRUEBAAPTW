Tablas en Bootstrap

Las tablas en Bootstrap sirven para organizar y presentar datos de manera clara y estructurada. Utilizando la etiqueta <table> con clases como .table, .table-striped, y .table-bordered, podemos mejorar el diseño y la legibilidad de nuestras tablas. Los elementos clave incluyen <thead> para el encabezado, <tbody> para el cuerpo, <tr> para las filas, y <th> para las celdas de encab
