Imágenes

En este apartado, exploraremos las capacidades de Bootstrap para trabajar con imágenes. Una de las características destacadas de Bootstrap es su capacidad para crear imágenes responsivas, lo que significa que las imágenes se ajustan automáticamente al tamaño de la pantalla del dispositivo en el que se están visualizando. Esto mejora significativamente la experiencia del usuario en diferentes dispositivos, desde teléfonos móviles hasta computadoras de escritorio.

Imagen con clase img-circle: Esta clase agrega un efecto de borde redondeado a la imagen, creando un círculo alrededor de ella.

Imagen con clase img-rounded: Esta clase agrega bordes suavemente redondeados a la imagen, lo que proporciona una apariencia más suave.

Imagen con clase img-thumbnail: Esta clase agrega un borde y un sombreado alrededor de la imagen, creando un efecto de miniatura.
